# Evaluador de Autómatas

Este proyecto es una aplicación que permite crear y evaluar autómatas finitos deterministas (AFD) y autómatas finitos no deterministas (AFND).

## Características

- Creación de autómatas: La aplicación permite al usuario definir estados, transiciones y estados finales para crear un autómata.
- Evaluación de cadenas: Una vez creado el autómata, el usuario puede ingresar una cadena para ser evaluada por el autómata.
- Visualización gráfica: Los autómatas se visualizan gráficamente en la aplicación, con los estados representados como círculos y las transiciones como flechas entre los círculos. Los estados finales se indican con un segundo círculo alrededor del estado, y el estado inicial se indica con un círculo azul.

## Uso

Para usar la aplicación, siga estos pasos:

1. Inicie la aplicación.
2. Use la interfaz de usuario para definir los estados, transiciones y estados finales de su autómata.
3. Ingrese una cadena para ser evaluada por el autómata.
4. La aplicación mostrará si la cadena es aceptada o rechazada por el autómata, y mostrará el camino que la cadena tomó a través del autómata.
